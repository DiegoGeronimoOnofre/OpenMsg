Este software na versão atual, funciona 
somente em redes internas, onde não se 
faz uso de NAT (ou tecnologias semelhantes)
como forma de comunicação.

Este aplicativo precisa ser executado
como administrador para que funcione corretamente.

Abaixo é listado as tecnologias para
o correto funcionamento dos aplicativos Servidor e Cliente.

Aplicativo servidor:
 * Java Standard Edition versão 6 ou superior
 * Firebird 2.1 Server Manager

Aplicativo cliente:
 * Java Standard Edition versão 6 ou superior

Não há garantia de funcionamento
dos aplicativos caso as versões das tecnologias 
instaladas sejam diferentes das tecnologias requeridas.
